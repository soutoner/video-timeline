## Preparations

### Requirements

`node, npm, git, make`

`npm install -g bower`

`npm install -g grunt-cli`

### Download dependencies

`npm install`

`bower install`

`cd bower_components`

`git clone https://github.com/mozilla/popcorn-js`

`cd popcorn-js`

`make`

### Compile SASS and CoffeeScript

`grunt`
